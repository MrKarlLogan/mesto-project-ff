# Проект: «Mesto»

Интерактивный сервис с пользовательскими карточками: добавление фотографий, лайки, удаление, редактирование профиля и просмотр изображений в модальных окнах. Проект реализован на чистом JavaScript с модульной архитектурой и сборкой через Webpack.

## Цели проекта
- Реализовать интерактивную работу с DOM на JavaScript
- Научиться создавать переиспользуемые компоненты и функции
- Освоить модульную архитектуру и `import / export`
- Настроить полноценную сборку проекта с помощью Webpack
- Реализовать модальные окна и пользовательские сценарии (CRUD)

## Используемые технологии

| Категория | Решение |
|---------|--------|
| Разметка | HTML5 |
| Стили | CSS3 |
| JavaScript | ES6+ |
| Архитектура | Модульный JS |
| Сборка | Webpack |
| Транспиляция | Babel |
| Пакетный менеджер | npm |
| Контроль версий | Git |
| Макет | Figma |

## Ключевые реализации

### Карточки
- Генерация карточек из массива данных
- Создание карточки через HTML `<template>`
- Удаление карточки по клику
- Лайк карточки с изменением состояния
- Добавление новых карточек через форму

### Модальные окна
- Универсальные функции открытия и закрытия попапов
- Закрытие по клику на оверлей
- Закрытие по клавише `Esc`
- Плавное открытие и закрытие через CSS-классы
- Отдельный попап для просмотра изображения

### Формы
- Редактирование профиля пользователя
- Предзаполнение полей текущими значениями
- Добавление карточки через форму
- Обработка `submit` с предотвращением стандартного поведения
- Очистка формы после отправки

### Архитектура и модульность
- Разделение логики по файлам:
  - `card.js` — создание карточек и обработчики лайков/удаления
  - `modal.js` — открытие и закрытие попапов
  - `index.js` — инициализация, DOM-элементы и обработчики событий
- Передача зависимостей через аргументы функций
- Отсутствие жёстких связей между модулями

### Сборка проекта
- Инициализация npm
- Две сборки: `dev` и `build`
- Минификация и транспиляция JavaScript
- Подключение CSS и ассетов через Webpack
- Исключение `node_modules` и `dist` из репозитория через `.gitignore`

## Структура проекта

| Путь | Назначение |
|----|-----------|
| `src/index.html` | Основная HTML-разметка |
| `src/index.js` | Точка входа, инициализация приложения |
| `src/components/card.js` | Логика карточек |
| `src/components/modal.js` | Логика попапов |
| `src/styles/` | CSS-файлы проекта |
| `dist/` | Собранная версия проекта |
| `package.json` | Скрипты и зависимости |

**Измеряется в:** реализованных пользовательских сценариях (CRUD карточек, попапы, формы), количестве независимых модулей и корректно настроенной сборке
